To-Do List App

To-Do List Application built using:

- Frontend → HTML, CSS, JavaScript
- Backend → Python Flask
- Database → MongoDB
- API Testing → REST API

Features

✅ Add Tasks  
✅ View Tasks  
✅ Update Tasks  
✅ Delete Tasks  
✅ MongoDB Database Connection  
✅ REST API Integration  
✅ Attractive UI Design  
✅ Full CRUD Operations

How It Works
User enters task in frontend.
JavaScript sends request to Flask API.
Flask backend stores task in MongoDB.
Tasks are fetched and displayed dynamically.
